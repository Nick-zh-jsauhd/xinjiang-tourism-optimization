## 最终模型口径

Q4-V2 将第四问建模为：

> 五一 12 天线路产品组合与分时容量优化问题：生成候选线路产品，选择投放组合，并在景区分时预约、酒店、车辆、导游、摆渡/停车等多资源容量下进行游客分配和策略仿真。

题面“游客人数与线路接待能力成比例”的假设通过 `strict_capacity_ratio_visitors` 保留为基准口径；现实偏好扩展与运营优化分别通过 `preference_elastic_visitors` 和 `balanced_optimized_visitors` 表示。

## 精确验证结果

Q4-V3 不重新生成线路，也不替代 Q4-V2 的分时容量仿真；它只在 Q4-V2 已生成的 63 条候选 12 天游线路产品中，保持 Q4-V2 的产品类型配额、区域覆盖、容量下界和质量代理约束，用 HiGHS-MILP 精确选择 18 条线路。

| 校验项 | Q4-V2 当前组合 | MILP 精确组合 |
|---|---:|---:|
| 求解状态 | - | optimal |
| 综合投放分 | 921.989 | 993.393 |
| 12天总容量 | 160190 人 | 187550 人 |
| 质量代理通过线路数 | 13 | 15 |
| 与当前组合重合 | - | 10/18 |

论文中应表述为“63条候选产品空间、固定投放政策下的精确最优组合”，不能表述为所有可能旅游线路的全局最优。
