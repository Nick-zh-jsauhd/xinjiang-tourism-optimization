## 最终模型口径

Q4-V2 将第四问建模为：

> 五一 12 天线路产品组合与分时容量优化问题：生成候选线路产品，选择投放组合，并在景区分时预约、酒店、车辆、导游、摆渡/停车等多资源容量下进行游客分配和策略仿真。

题面“游客人数与线路接待能力成比例”的假设通过 `strict_capacity_ratio_visitors` 保留为基准口径；现实偏好扩展与运营优化分别通过 `preference_elastic_visitors` 和 `balanced_optimized_visitors` 表示。

| 极端补运力策略可接待 | 143583 人 |

第四问结果应同时解释 `policy_pass` 与 `strict_policy_pass`：服务率高不代表所有时段都不拥挤，严格通过才表示景区分时段和多资源均完全可控。
