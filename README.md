## 关键结果

### 第一问

最终材料位于：

```text
第一问_30天路线优化材料包/09_Q1_V3_鲁棒联合优化/
第一问_30天路线优化材料包/10_Q1_V4_精确求解器验证/
第一问_30天路线优化材料包/11_Q1_V5_题面优先成本覆盖校准/
```

Q1-V3 不把问题简化为 TSP，而是考虑覆盖、费用、时间窗、缓冲日、舒适度、红色压力日和扰动风险。32 景点路线作为覆盖上界，24 景点方案作为运营鲁棒主推。

Q1-V4 在固定 `Q1V3_Q24_120` 的24个景点和固定首尾点下，用 HiGHS-MILP 精确求解交通时间最小开放路径。V3 顺序 best-label 交通时间为 80.339 小时，MILP 精确排序为 76.620 小时，相对差距 4.8538%。

Q1-V5 按题面“花最少的钱游尽可能多的地方”重新校准求解叙事：30天严格可执行的最高覆盖候选为 `Q1V3_Q28_269`，覆盖28景点、费用13022.84元；加入运营成功率约束后的方案为 `Q1V3_Q26_215`，覆盖26景点、费用10557.60元、运营成功率0.862；原 `Q1V3_Q24_120` 继续作为现实鲁棒主推保留。

### 第二问

最终材料位于：

```text
第二问_两年暑假交通费用优化材料包/11_Q2_V3_两年鲁棒多模式路径覆盖/
第二问_两年暑假交通费用优化材料包/12_Q2_V4_精确求解器验证/
```

Q2-V3 以两年覆盖普通游客可达景点为约束，最小化新疆境内交通费用，并比较固定乌鲁木齐起讫、开放多口岸费用下界和鲁棒综合主推方案。

Q2-V4 对20节点两年覆盖代表性子问题建立两条乌鲁木齐起讫路径 MILP，约束每年 8-12 个景点并由 HiGHS 证明最优。原抽样顺序费用为 2594.34 元，MILP 精确费用为 1681.10 元，说明后续若继续强化第二问，应优先推进年度分组与路线排序联合优化。

### 第三问

最终材料位于：

```text
第三问_文化考察均衡优化材料包/10_Q3_V2_鲁棒三团队文化考察调度/
第三问_文化考察均衡优化材料包/11_Q3_V3_精确求解器验证/
```

Q3-V3 基于 Q3-V2 候选点和交通标签，建立 Held-Karp 子集路径 + HiGHS-MILP 集合划分模型。在楼兰古城和尼雅遗址固定由专项审批组执行、三组均乌鲁木齐起讫、资源不跨组共享的政策空间内，最大完成时间为 99.187 小时，组间差距为 9.994 小时，gap=0。

### 第四问

最终材料位于：

```text
第四问_五一容量接待优化材料包/09_Q4_V2_12天线路产品组合与分时容量优化/
第四问_五一容量接待优化材料包/10_Q4_V3_精确求解器验证/
```

Q4-V2 将节假日游客接待建模为 12 天线路产品组合与多资源分时容量优化。最终生成 63 条候选线路，入选 18 条，12 天总容量 160190 人，并保留题面容量比例分配、现实偏好弹性分配和运营优化分配三种口径。

Q4-V3 在63条候选线路、固定产品类型配额、区域覆盖、容量下界和质量代理约束下，用 HiGHS-MILP 精确选择18条线路。相对 Q4-V2 当前组合，综合投放分从 921.989 提升到 993.393，12天容量从 160190 人提升到 187550 人。

## 精确性声明

Q1-V4、Q2-V4、Q3-V3、Q4-V3 的“optimal/gap=0”只适用于各自明确限定的精确子问题或候选空间。第一问、第二问和第四问的完整问题仍包含候选生成、时间窗修复、鲁棒仿真和策略评估，不能在论文中声称完整全局最优。
