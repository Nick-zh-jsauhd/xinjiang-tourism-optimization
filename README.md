## 关键结果

### 第一问

最终材料位于：

```text
第一问_30天路线优化材料包/09_Q1_V3_鲁棒联合优化/
```

Q1-V3 不把问题简化为 TSP，而是考虑覆盖、费用、时间窗、缓冲日、舒适度、红色压力日和扰动风险。32 景点路线作为覆盖上界，24 景点方案作为运营鲁棒主推。

### 第二问

最终材料位于：

```text
第二问_两年暑假交通费用优化材料包/11_Q2_V3_两年鲁棒多模式路径覆盖/
```

Q2-V3 以两年覆盖普通游客可达景点为约束，最小化新疆境内交通费用，并比较固定乌鲁木齐起讫、开放多口岸费用下界和鲁棒综合主推方案。

### 第三问

最终材料位于：

```text
第三问_文化考察均衡优化材料包/10_Q3_V2_鲁棒三团队文化考察调度/
第三问_文化考察均衡优化材料包/11_Q3_V3_精确求解器验证/
```

Q3-V3 基于 Q3-V2 候选点和交通标签，建立 Held-Karp 子集路径 + HiGHS-MILP 集合划分模型。在楼兰古城和尼雅遗址固定由专项审批组执行、三组均乌鲁木齐起讫、资源不跨组共享的政策空间内，最大完成时间为 99.187 小时，组间差距为 9.994 小时，gap=0。

### 第四问

最终材料位于：

```text
第四问_五一容量接待优化材料包/09_Q4_V2_12天线路产品组合与分时容量优化/
```

Q4-V2 将节假日游客接待建模为 12 天线路产品组合与多资源分时容量优化。最终生成 63 条候选线路，入选 18 条，12 天总容量 160190 人，并保留题面容量比例分配、现实偏好弹性分配和运营优化分配三种口径。

## 复现说明

每一问材料包内均保留：

- `00_...`：题目与建模口径；
- `01_输入数据/`：该问复现所需数据；

- `README.md`、`*_file_manifest.csv`、`*_directory_summary.csv`。

论文与答辩应引用各问最终目录中的输出，不再引用已删除的 V1/中间探索结果。
