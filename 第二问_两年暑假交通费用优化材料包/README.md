## 最终模型口径

Q2-V3 将第二问建模为：

> 两年暑假多模式路径覆盖与新疆境内交通费用最小化问题：在两年完成普通游客可达景点覆盖的前提下，比较固定乌鲁木齐起讫、开放多口岸费用下界和鲁棒综合主推方案。

模型已修正旧版口径问题：`scenic_shuttle` 不再以 0 元参与费用最小化，`self_drive` 统一规范为 `taxi_transfer / rental_car / charter_car` 等现实交通费用模式。

## 核心结果

| 方案 | 解释 |
|---|---|
| 保守执行方案 | 固定乌鲁木齐起讫，覆盖 38 个普通游客可达景点，运营成功率高 |
| 境内费用下界方案 | 开放式多口岸，境内交通费最低，但两年负担不均衡 |
| 鲁棒综合主推方案 | 开放式多口岸，兼顾境内交通费、年度均衡、舒适度与鲁棒性 |

Q2-V3 输出摘要显示：

- 普通游客可达景点：38 个
- 多模式标签：2845 条
- 网关标签：800 条
- 候选计划：89 个
- 入选计划：6 个
- Pareto 前沿：15 个

## 精确验证结果

Q2-V4 补上原 Q2-V3 中未运行的 20 节点两年覆盖 MILP。该实验从固定乌鲁木齐主方案中抽取 Year1 前10点和 Year2 前10点，精确求解两条乌鲁木齐起讫路线覆盖20个景点，并约束每年 8-12 个景点。

| 校验项 | 结果 |
|---|---:|
| 求解状态 | optimal |
| 原抽样顺序费用 | 2594.34 元 |
| MILP 精确费用 | 1681.10 元 |
| 相对差距 | 54.3240% |

这个差距说明第二问若继续追求更强求解质量，应优先把“年度分组 + 路线排序”做成联合优化。论文中应表述为“20节点代表性子问题精确最优”，不能表述为完整38景点两年鲁棒模型全局最优。
