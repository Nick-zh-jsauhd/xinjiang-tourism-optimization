# 可视化数据底座

本目录由 `../scripts/build_visual_data.py` 自动生成。

这些 CSV 是制图输入层，不是新的模型实验结果。它们只做字段筛选、格式转换、派生分类、长宽表转换和图表排序。

## 主要输出

| 文件 | 用途 |
|---|---|
| `overview_model_evolution.csv` | 四问模型演化总览图 |
| `q1_visual_route_tiers.csv` | Q1 方案漏斗和路线层级说明 |
| `q1_visual_success_comparison.csv` | Q1 运营成功率 vs 严格舒适成功率 |
| `q1_visual_daily_pressure.csv` | Q1 每日压力条带图 |
| `q2_visual_plan_compare.csv` | Q2 三类代表方案费用/鲁棒性对比 |
| `q2_visual_year_balance.csv` | Q2 年度负担均衡图 |
| `q2_visual_mode_mix.csv` | Q2 交通方式构成图 |
| `q2_visual_gateway_threshold.csv` | Q2 多口岸外部差价阈值图 |
| `q3_visual_group_summary.csv` | Q3 三组完成时间图 |
| `q3_visual_buffer_policy_long.csv` | Q3 缓冲策略离散点图 |
| `q3_visual_exact_check_card.csv` | Q3 固定政策空间精确校验卡片 |
| `q4_visual_capacity_compare.csv` | Q4 旧9线路 vs Q4-V2容量提升 |
| `q4_visual_quality_audit.csv` | Q4 线路质量审计图 |
| `q4_visual_strategy_matrix.csv` | Q4 需求冲击-策略表现热力图 |
| `q4_visual_timeslot_pressure.csv` | Q4 景区分时压力热力图 |
| `q4_visual_shadow_top10.csv` | Q4 瓶颈影子价格 Top 10 |
| `q4_visual_allocation_comparison_long.csv` | Q4 三种分配口径对比 |
