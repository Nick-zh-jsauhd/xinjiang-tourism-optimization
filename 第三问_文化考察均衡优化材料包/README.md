# 第三问：三组文化考察均衡优化材料包

本材料包已完成清理，仅保留题目口径、输入数据、Q3-V2 业务主模型和 Q3-V3 精确求解器验证。旧版图论基线、早期 MinMax 探索、旧报告和旧图表目录已删除。

第三问的真实决策对象不是普通游客路线，而是“少数民族研究所三组文化考察队如何并行完成新疆文化点考察”。因此模型主线应表述为：

> 三团队文化考察 MinMax 调度：在特殊准入、现场考察时长、安全缓冲和交通时间约束下，最小化三组最大完成时间，并兼顾组间任务均衡。

## 最终模型口径

- Q3-V2 是业务主模型：鲁棒三团队文化考察调度，包含文化候选点、特殊准入、三组 MinMax 分配、日级排程、资源日历和缓冲策略仿真。
- Q3-V3 是精确求解器验证：在 Q3-V2 的候选点和交通标签基础上，建立 Held-Karp 子集路径 + HiGHS-MILP 集合划分模型，验证当前分组在固定政策空间内精确最优。
